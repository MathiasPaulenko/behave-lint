# Benchmarks

Performance benchmark results and fixtures for `behave-lint`.

See [REPORT.md](REPORT.md) for the latest benchmark results and analysis.

## Structure

- `fixtures/` — Generated .feature files for benchmarking
- `results/` — Benchmark result snapshots (generated by nightly CI)
- `REPORT.md` — Latest performance report

## Running benchmarks

```bash
# Generate fixtures (10, 100, or 1000 files)
python scripts/generate_fixtures.py --count 100 --output benchmarks/fixtures

# Run all performance tests
pytest -m performance -v

# Run a specific benchmark
pytest tests/performance/test_benchmarks.py::test_perf_100_files -v
```

## Performance targets

| Files | Cross-platform | CI (Ubuntu) |
|-------|---------------|-------------|
| 10    | <1s           | <0.5s       |
| 100   | <2s           | <1s         |
| 1000  | <15s          | <5s         |
| 5000  | —             | <30s        |
